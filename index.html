Selected Publications

  • AliO: Output Alignment Matters in Long-Term Time Series Forecasing NeurIPS 2025

    Proposed a novel loss function (AliO) to address the output alignment problem in Long-Term Time Series Forecasting (LTSF). Achieved up to 27.5% MSE improvement and 58.2% TAM improvement across various models and datasets.

  • PPT: Patch Order Do Matters In Time Series Pretext Task ICLR 2025

    Introduced Patch order-aware Pretext Task (PPT), the first order-aware self-supervised learning task for patch-based time series models. Achieved up to 7% accuracy improvement in supervised cardiogram tasks, enhancing model performance by integrating structural order awareness.

  • Smart ECU: Scalable On-Vehicle Deployment of Drivetrain Fault Classification Systems for Commercial Electric Vehicles CIKM 2025

    Pioneered the first on-vehicle drivetrain fault classification system (Smart ECU) deployed on a resource-constrained MCU for commercial EVs. Achieved 99.24% memory reduction compared to prior cloud-based work while maintaining high diagnostic accuracy.

  • SMMF: Square-Matricized Momentum Factorization for Memory-Efficient Optimization AAAI 2025

    Developed SMMF, a memory-efficient optimizer that reduces the memory requirement of adaptive optimizers (e.g., Adam) by up to 96% using a novel square-matricization and one-time factorization scheme. Confirmed theoretical convergence via regret bound analysis.
